Command Line Interface

The Language Detector also works as a command-line application, allowing you to detect languages directly from your terminal.

How to Use the CLI Version:
$ python language_detector.py

When you run the CLI version:

  1. You'll be prompted to enter text
  2. The application will detect the language
  3. The detected language and its code will be displayed
  4. Enter 'q' at any time to quit the application
Example Usage:
===== Language Detector =====
Enter text to detect its language (or 'q' to quit)

Enter text: Hello, how are you doing today?

Detected Language: English (en)

Enter text: Hola, ¿cómo estás?

Detected Language: Spanish (es)

Enter text: q
Goodbye!
Tip: For the most accurate results, provide a few sentences in the language you want to detect.

Technical Details

The language detection is powered by the langdetect Python library, which implements a Naive Bayes classifier to identify languages.

Key Features:
  • Fast and accurate language detection
  • Support for 55+ languages
  • Works with short phrases and longer texts
  • Built-in error handling for edge cases
Requirements:
  • Python 3.x
  • langdetect library
This project demonstrates Natural Language Processing (NLP) techniques in a practical application.
